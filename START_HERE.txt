1. START THE APPLICATION
   - Run startup scripts (see above)
   - Backend runs on: http://localhost:8080
   - Frontend runs on: http://localhost:3000

2. REGISTER USERS
   - Open http://localhost:3000 in browser
   - Enter username when prompted
   - You'll be registered as User 1

3. CREATE SECOND USER
   - Open another browser window/tab
   - Enter different username
   - You'll be registered as User 2

4. MAKE A CALL
   - In User 1 window: Select User 2 from "Available Users"
   - Click "Call" button
   - User 1 status changes to "CALLING" (yellow, pulsing)

5. ACCEPT CALL
   - In User 2 window: Modal appears "Call from: User1"
   - Click "Accept" button
   - Both users change to "IN CALL" (green, pulsing)
   - Call timer starts counting

6. CONTROL THE CALL
   - Click "Mute" to toggle microphone on/off
   - Click "Hold" to pause call (status â†’ "ON HOLD")
   - Click "Resume" to continue
   - Watch audio visualization animate
   - Call timer counts up

7. END THE CALL
   - Click "End Call" button
   - Both return to "IDLE" status
   - Can make new calls immediately
